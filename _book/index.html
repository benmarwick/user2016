1 Introduction

This acts as a repository for my favourite video talks from the recent useR2016 conference along with the ability to view any of the talks via a clickable table. It is almost certainly not the best method of presentation but is a trial run for creating and deploying interactive books to bookdown.org

Author

Andrew Clark is an independent R developer based in North Vancouver He has for many years supplied statistical sports data on the web but with the interactive opportunities arising from the shiny framework and the ability to swiftly create interesting visualizations via dashboards has branched out into many different areas. An introduction to his work can be found under the mytinyshinys umbrella
